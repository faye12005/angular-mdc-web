Lists

MDC List provides styles which implement Material Design Lists - "A single continuous column of tessellated subdivisions of equal width." Both single-line and two-line lists are supported.
Component
mdc-list
Properties Description
dense: boolean Increases the density of the list, making it appear more compact.
lines: number (Set to 2 for two line list. Default is 1) Increases the height of the row to give it greater visual separation from adjacent rows.
avatar: boolean Configures the leading tiles of each row to display images instead of icons.
border: boolean Sets a border around each list item.
interactive: boolean List items receive styles for hover, focus, and press states (including the ripple). Default is true.
Event Description
MdcListSelectionChange(source: MdcListItem) Change event that is being fired whenever the selected state of an option changes.
Directive
mdc-list-item
a[mdc-list-item]
Properties Description
selected: boolean Whether or not the item is selected. Setting this sets the selected item.
Methods Parameters
toggle() Toggles the selection state of the option.
Child Directive
mdc-list-item-graphic The first tile in the row (in LTR languages, the left-most).
mdc-list-item-meta The last tile in the row (in LTR languages, the right-most)
mdc-list-item-text Primary text for the row (displayed as the middle tile)
mdc-list-item-secondary Secondary text for the row (displayed in the middle tile)
Directive
mdc-list-divider Separates two rows or two lists using a horizontal bar.
Properties Description
inset: boolean Increases the leading margin of the divider so that it does not intersect the avatar column.
padded: boolean To make a divider match the padding of list items.
Directive
mdc-list-group Wraps one or more mdc-list and/or mdc-list-divider elements.
mdc-list-group-subheader Heading text displayed above each list in a group.
